English

The World Database on Protected Areas (WDPA) User Manual provides information and guidance about the data held within the WDPA, including its history, how it is collected, managed and distributed, and how it should be interpreted and used for analyses and research.

The Manual has been prepared for WDPA data providers and users. It is structured in 4 sections and includes 6 appendices. The WDPA user manual is a dynamic and evolving document. It is recommended that all parties interested in using the WDPA read these guidelines prior to using the WDPA for any analyses that will inform research or decision making. For any queries regarding collation, use, processing of the database, or any feedback regarding this manual please contact protectedareas@unep-wcmc.org.

Pусский

Всемирная база данных по охраняемым природным территориям (ВБДОПТ) является продуктом совместного сотрудничества ООН окружающая среда и Международного Союза по Охране Природы (МСОП). Её составил Всемирный центр мониторинга природоохраны ООН окружающая среда (UNEP-WCMC), с помощью правительств, неправительственных организаций, университетов и промышленного сектора. ВБДОПТ является обширной всемирной базой данных для морских и сухопутных защищенных територий, содержащая и пространственную информацию (границы и точки) и их свойства (к примеру, информация в виде таблиц). Базу данных можно найти он-лайн на Охраняемой Планете (www.protectedplanet.net), где её можно и просмотреть и скачать.
